City Planner Tool

My final project for the data engineering and analytics course for Brights Training.
The goal of this project was to create a tool for city planners in Finland and Sweden. This tool will help city planners with reducing emissions by knowing the emission output for the city and by planting recommended amount of trees to neutralize emissions.
We worked as a team of 4 with machine learning, ETL processing, postgresql database and flask web application powered with pyplot graphs.
